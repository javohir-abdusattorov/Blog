# Blog

Asosan Vue.js'ga e'tibor berilgan blog proyekt.

## Technologies

**Library**: Express.js \
**Database**: MongoDB \
**ORM**: Mongoose \
**Frontend**: Vue.js \
**Other**: Vuex, Vue Router

## Installation

Dependencies:
- Node.js
- Npm
- MongoDB
- Vue.js, Vue cli (For editing frontend)

**1. Start backend**

```bash
cd Backend
npm install
node server
```

**1. Start frontend**

```bash
cd Frontend/production
npm install
node server
```


**Backend**: [localhost:5000](http://localhost:5000) \
**Frontend**: [localhost:3000](http://localhost:3000)
